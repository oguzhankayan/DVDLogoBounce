# Security Policy

Corner is a local-first tvOS app. It has no account system, analytics SDK, backend service, or network calls in the current source tree.

## Reporting a Vulnerability

Please report security issues privately by email:

`facebookhesap006@gmail.com`

Include a clear description, reproduction steps, affected files or versions, and any proof of concept that helps verify the issue. Please do not open a public GitHub issue for a vulnerability until it has been reviewed.

## Supported Versions

This repository is maintained as source code. Security fixes are made on the default branch unless release branches are introduced later.
